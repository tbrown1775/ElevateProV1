# ElevatePro AI Operating Rules

This repository is the production-track source of truth for Elevate. These rules apply to every AI and human contributor.

## Authority order
1. Explicit product-owner decisions.
2. Approved requirements in `docs/requirements/`.
3. Approved architecture and ADRs in `docs/architecture/` and `docs/decisions/`.
4. The active GitHub issue and its acceptance criteria.
5. Existing implementation patterns
